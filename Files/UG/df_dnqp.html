$${\delta f \over \delta n_{qp}} = -{\alpha \gamma \over 2}\cdot {f_0\over \sigma_2} \cdot {d\sigma_2\over dn_{qp}}.$$

I'll use standard numerical values for Al: $f_0 = \omega_0/2\pi = 1$GHz, $\alpha = .1$, $T_c = 1.2$K, $\Delta_0 = 1.76kT_c$, $T = 10$mK, $\gamma = -1/2$, and work constructively towards the final expression.

def sigma_2(T, Delta0, omega):
    xi = hbar*omega/(k*T)
    #print('hbar omega/kT = ', xi)
    second_term = np.sqrt(2.*pi*k*T/Delta0)*np.exp(-Delta0/(k*T))
    
    third_term = 2.*np.exp(-Delta0/(k*T))*np.exp(-xi)*special.i0(xi)
    
    result =  (pi*Delta0/(hbar*omega))*(1.-second_term - third_term)
    #print('sigma2 = ',result)
    return result

From PB Thesis (and Gao): $\Delta(T) \approx \Delta_0\left(1 - {n_{qp}\over 2N_0\Delta(T)}\right).$

This implicitly defines an equation for $\Delta(T)$, and I want to find that equation.

We start by making a quadratic: $\Delta(T)^2 = \Delta_0(\Delta(T) - {n_{qp}\over 2N_0}) \Rightarrow \Delta(T)^2 - \Delta_0\Delta(T) + {n_{qp}\Delta_0\over 2N_0} = 0,$

and we use the quadratic formula to obtain

$$\Delta(T) = {\Delta_0 \over 2} \pm {\sqrt{\Delta_0^2 - 2{n_{qp} \Delta_0\over N_0}}\over 2}.$$

I want to expand the square root term when $n_{qp}/N_0 \ll 1$.

$$ \sqrt{\Delta_0^2 - 2{n_{qp} \Delta_0\over N_0}} = \Delta_0\sqrt{1 - 2{n_{qp}\over N_0\Delta_0}} \approx \Delta_0\left(1 - {n_{qp} \over N_0\Delta_0}\right), $$

which shares the units of the original approximation.

Then, ignoring the subtracting solution, we have $\Delta(T) = \Delta_0\left(1 - {2n_{qp}\over N_0\Delta_0} \right)$, which is intuitive given the original approximation.

PB Thesis gives $$\tau_{qp} = {\tau_0\over\sqrt{\pi}} \cdot \left(kT_c\over 2\Delta(T)\right)^{5/2}\sqrt{T_c\over T}e^{\Delta(T)\over kT} = {1\over Rn_{qp}},$$

where $$R^{-1}=N_0\tau_0\cdot {(kT_c)^3\over 2\Delta(T)^2},$$ the units of $R$ being m$^3$s$^{-1}$. $\tau_0$ is a material parameter. Then, we have $N_{qp}\tau_{qp} = {V\over R}$ has units of seconds, which is the quantity we wish to measure.

The problem is that there are not enough equations to fit for $R$ and $\tau_0$ simultaneously, so I'll estimate $\tau_0 = 2.5 \times 10^{-6}$ seconds as the value that causes $1/Rn_{qp}$ to agree with $\tau_{qp}$.

def nqp(T):
    return 2*N0*np.sqrt(2*pi*k*T*Delta0)*np.exp(-Delta0/(k*T))

def Delta(nqp):
    return Delta0*(1 - (2.*nqp/(N0*Delta0)))

def tau_qp(T, tau0):
    n_qp = nqp(T)
    delta = Delta(n_qp)
    return (tau0/np.sqrt(pi) * (k*Tc/(2.*delta))**(5./2.)*np.sqrt(Tc/T)*np.exp(delta/(k*T)))
